*Make sure you have PyTK mod (https://www.nexusmods.com/stardewvalley/mods/1726)*

If you have DCBurger's high res portrait mod, copy the contents in my "Default" folder (both the png and the .pytk file) into Stardew Valley\Mods\[CP] DCBurger's High Res Portraits\assets folder to overwrite the assets in it to install.

If you have SVE, copy the contents in my "SVE" folder (both the png and the .pytk file) into Stardew Valley\Mods\Stardew Valley Expanded\[CP] Stardew Valley Expanded\assets\CharacterFiles\Portraits\Magnus folder to overwrite the assets in it to install.

If you have lunakatt's DCBurger style portraits for SVE, copy the contents in my "SVE" folder (both the png and the .pytk file) into Stardew Valley\Mods\[CP] lunakatt's DCBurger style portraits for SVE\assets folder to overwrite the assets in it to install.
